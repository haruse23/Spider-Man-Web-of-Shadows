# Spider-Man-Web-of-Shadows
Tools for the video-game Spider-Man: Web of Shadows

Use the provided `cli.py` and `pcapk.py` files instead of the original ones along with the `CLI Tool` to generate `.standalone` files and `.standalone_mesh` files from the `PCPACK` file

`CLI Tool`:
https://github.com/kirbystealer/WebOfShadowsTools/blob/master/python

Original `cli.py` and `pcapk.py`:
